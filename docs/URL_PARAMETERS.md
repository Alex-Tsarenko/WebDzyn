# URL-параметры для присоединения к комнате

## Использование

Теперь можно присоединиться к комнате, передав её ID в URL-параметре.

### Формат URL

```
http://localhost:3000/?room=ROOMID
```

Где `ROOMID` - это 6-значный ID комнаты.

## Примеры использования

### Пример 1: Прямая ссылка на комнату

```
http://localhost:3000/?room=ABC123
```

При открытии этой ссылки:
1. ✅ ID комнаты автоматически подставится в поле ввода
2. ✅ Появится уведомление "Присоединение к комнате ABC123..."
3. ✅ Пользователю нужно только нажать кнопку "Присоединиться"

### Пример 2: Поделиться комнатой

После создания комнаты:
1. URL автоматически обновится: `http://localhost:3000/?room=XYZ789`
2. Скопируйте этот URL и отправьте собеседнику
3. Собеседник откроет ссылку и сразу увидит ID комнаты

### Пример 3: Создание постоянной ссылки

Вы можете создать постоянную ссылку для быстрого доступа:

```html
<a href="http://localhost:3000/?room=MYROOM">Присоединиться к звонку</a>
```

## Как это работает

### 1. Автоматическое заполнение поля

При загрузке страницы функция `checkUrlParameters()` проверяет URL:

```javascript
function checkUrlParameters() {
  const urlParams = new URLSearchParams(window.location.search);
  const roomId = urlParams.get('room');
  
  if (roomId) {
    elements.roomIdInput.value = roomId.toUpperCase();
    showNotification(`Присоединение к комнате ${roomId.toUpperCase()}...`, 'success');
  }
}
```

### 2. Обновление URL при создании комнаты

Когда вы создаёте комнату, URL автоматически обновляется:

```javascript
function updateUrlWithRoom(roomId) {
  const url = new URL(window.location);
  url.searchParams.set('room', roomId);
  window.history.pushState({}, '', url);
}
```

## Преимущества

✅ **Удобство**: Можно поделиться прямой ссылкой на комнату  
✅ **Быстрота**: Не нужно вручную вводить ID комнаты  
✅ **Интеграция**: Можно встроить ссылки в другие приложения  
✅ **История**: URL сохраняется в истории браузера

## Безопасность

⚠️ **Важно**: ID комнаты виден в URL, поэтому:

1. Не используйте предсказуемые ID
2. Комнаты автоматически удаляются, когда все пользователи выходят
3. Максимум 2 пользователя в комнате
4. Для production рекомендуется добавить аутентификацию

## Дополнительные параметры (будущие возможности)

Можно расширить функционал, добавив другие параметры:

```
http://localhost:3000/?room=ABC123&name=Иван&muted=true
```

### Возможные параметры:

- `name` - имя пользователя
- `muted` - начать с выключенным микрофоном
- `autoJoin` - автоматически присоединиться без нажатия кнопки
- `theme` - выбор темы оформления

### Тест 1: Создание комнаты
1. Откройте `http://localhost:3000`

3. ✅ Должно появиться уведомление

### Тест 3: Копирование ссылки

2. Скопируйте URL из адресной строки

## Отладка

Проверить параметры URL в консоли:

```javascript
// Получить текущие параметры
const urlParams = new URLSearchParams(window.location.search);
console.log('Room ID:', urlParams.get('room'));

// Установить новый параметр
const url = new URL(window.location);
url.searchParams.set('room', 'TEST01');
console.log('New URL:', url.toString());
```
